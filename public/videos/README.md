# Videos Directory

This directory stores the generated video for the Virtual Tour section.

## Generated Video

- **File**: `innoson-g80-highway.mp4`
- **Description**: High-quality cinematic video of Innoson G80 driving on a scenic highway
- **Generated by**: Veo 3 AI model
- **Purpose**: Default video for the Virtual Tour section

## Generating the Video

The video is generated once and saved here. To generate it:

1. **Automatic**: Click "Generate Video" button in the Virtual Tour section
2. **Manual API Call**: 
   ```bash
   curl -X POST http://localhost:3000/api/gemini/generate-and-save
   ```

## Notes

- The video is generated using Veo 3 and may take several minutes
- Once generated, it will be reused automatically
- To regenerate, delete the existing file and generate again
- The video is served from `/videos/innoson-g80-highway.mp4`
